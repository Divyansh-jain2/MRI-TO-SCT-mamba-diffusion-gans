MambaBlock3D — Inner Mechanism (shared by both models)
01
Flatten Spatial
3D volume reshaped into a 1D token sequence. Each voxel position becomes one token.
(B,C,D,H,W)
→ (B, D·H·W, C)
02
LayerNorm
Normalize each token's features for stable training.
x_norm = LayerNorm(x_flat)
03
Selective SSM
Mamba processes the sequence. Input-dependent gates decide what to remember vs. forget — linear O(N) cost.
h(t) = A·h(t-1) + B·x(t)
y(t) = C·h(t)
A,B,C = f(input)
04
Linear Projection
Map SSM output back to channel dimension via a learned linear layer.
x_out = Linear(x_ssm)
05
Residual Add
Add original input (before SSM) to the output. Gradient highway for deep networks.
x_out = x_out + residual
06
Reshape to 3D
Unflatten the sequence back to the original 3D spatial volume layout.
(B, D·H·W, C)
→ (B,C,D,H,W)
