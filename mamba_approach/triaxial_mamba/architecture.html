■ Component Deep Dives

TriAxial Bidirectional Scanning Core SSM

The old MambaBlock3D flattened (B,C,D,H,W) → (B, D·H·W, C) creating 524K-token sequences that destroyed spatial locality.

TriAxial scans along 3 axes × 2 directions = 6 SSM passes:

  • D-axis: (B·H·W, D, C) — max 32 tokens
  • H-axis: (B·D·W, H, C) — max 128 tokens
  • W-axis: (B·D·H, W, C) — max 128 tokens

Outputs fused via 1×1×1 Conv (3C→C) + residual.

CBAM3D Attention Skip Gates

Applied to encoder skip connections before decoder concatenation. Filters out noisy/irrelevant features.

Channel Attention: GlobalAvgPool + GlobalMaxPool → shared MLP → sigmoid gate per channel.

Spatial Attention: Channel-wise avg+max → Conv3d(2→1) → sigmoid gate per voxel.

Cost: negligible (~1K params each).

V2 Loss Function 3-Stage

Stage 1 (ep 0–49): WeightedMAE only (warmup)

Stage 2 (ep 50–199): wMAE + SSIM + GradientLoss

Stage 3 (ep 200+): + FocalFrequencyLoss

L = 1.0·wMAE + 0.2·SSIM + 0.05·Grad + 0.1·FFL
+ 0.4·DS₂_loss + 0.2·DS₃_loss (deep sup)

wMAE weights: bone(HU>300)=3×, soft=1.5×, air=0.5×

GradientLoss Edge

Computes finite differences along D, H, W axes for both prediction and target, then penalizes the difference.

L_grad = ⅓ · Σ_axis |∇pred - ∇target|

Critical for sharp bone boundaries that L1 smooths out.

FocalFrequencyLoss FFT

Converts pred/target to frequency domain via 3D rFFT. Adaptively weights frequencies where the error is largest.

L_ffl = E[ w(f) · |F(pred) - F(target)| ]
w(f) = |F(pred)-F(target)|^α / mean

Recovers high-frequency textures that standard L1/SSIM miss. (ICCV 2021)

Memory Optimizations VRAM

  • Gradient Checkpointing on all 8 TriMambaEncoderBlocks — recomputes activations during backward, saves ~60% VRAM
  • AMP (fp16) — halves activation memory
  • Sequential axis processing with explicit del statements
  • zero_grad(set_to_none=True) — avoids gradient zero-fill
  • Aggressive freeing of encoder features in decoder
■ Data Pipeline

Input Format

Each .npy file: shape (2, 192, 192, 96)

  • Channel 0: T2 MRI (normalized [-1, 1])
  • Channel 1: CT ground truth (normalized [-1, 1])
  • HU clip range: [-1024, 1500]

Training Augmentation

  • Random 3D patch extraction (32×128×128)
  • Random flip along D axis (50%)
  • Random flip along H axis (50%)
  • Random flip along W axis (50%)
  • Intensity jitter ±0.03 on MRI (50%)

Data Splits

Directory: /DATA/divyansh/mc_ddpm_data/brain_npy/

  • imagesTr/ — training set
  • imagesVal/ — validation set
  • imagesTs/ — test set

Val/Test use full volume (96×192×192), no patching.

■ Training Timeline

Phase 1: Warmup (ep 0–49)

  • Loss: WeightedMAE only
  • LR: cosine from 2e-4
  • No EMA, no SSIM/Grad losses
  • Model learns coarse intensity mapping

Phase 2: Refinement (ep 50–199)

  • Loss: wMAE + SSIM + GradientLoss
  • EMA tracking begins (decay=0.9999)
  • Deep supervision activated (aux heads)
  • Model learns structure + edges

Phase 3: Fine-tuning (ep 200–500)

  • Loss: + FocalFrequencyLoss
  • High-frequency detail recovery
  • Bone boundary refinement
  • Best model saved by lowest val MAE
